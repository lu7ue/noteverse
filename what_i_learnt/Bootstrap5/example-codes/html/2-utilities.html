导航栏 Navbar

Bootstrap 的导航栏由 navbar 类控制，通常包含品牌、折叠按钮和菜单项。

  • navbar-expand-lg：在大屏以上显示横向菜单，小屏时折叠成汉堡按钮。
  • 折叠按钮 navbar-toggler 配合 collapse navbar-collapse 使用，实现响应式折叠菜单。
  • ms-auto：将菜单项靠右对齐。

表单 Forms（全面示例）

涵盖输入框、下拉框、单复选框、文本区、按钮和输入组。

文本输入框/文本区都使用 form-control，下拉框使用 form-select。

提示框 Alert

Bootstrap 提供内置提示框类，用于显示不同类型的消息或状态。

使用 alert 类创建提示框，再加上颜色类 alert-primary ~ alert-dark。
role="alert" 用于辅助无障碍提示，屏幕阅读器会识别它。
可搭配 alert-dismissible 和按钮实现可关闭提示框。

面包屑导航 Breadcrumb

Bootstrap 提供内置面包屑类，用于显示页面层级结构，帮助用户了解当前位置。面包屑本质是一个有序列表，内置样式自动添加分隔符 /。

使用 breadcrumb 类创建面包屑容器，breadcrumb-item 创建每一层导航。
当前页添加 active 类并使用 aria-current="page"，屏幕阅读器可以识别当前页面。
适合用于多层页面结构，快速展示用户所在位置。

列表组 List group

Bootstrap 提供列表组组件，用于显示整齐的列表项，支持颜色类、激活状态（用 active 来标识“当前项”，视觉高亮）和可点击项（list-group-item-action 提供鼠标悬停和点击反馈，使列表项可交互），常用于菜单、状态列表、信息展示或导航。

list-group 用于容器，list-group-item 用于每个列表项。
可加 list-group-item-* 颜色类显示不同状态信息。
列表组可嵌套链接或按钮，使列表可点击。

进度条 Progress

Bootstrap 提供内置进度条类，用于显示任务或操作的完成进度。

内置样式自动显示带圆角轨道，支持多种颜色表示状态，可自定义宽度显示完成百分比，可叠加多条 progress-bar 显示复杂进度。

progress 是外层容器，表示进度条轨道。
progress-bar 是实际进度条，bg-* 控制颜色。
通过 style="width: %" 设置进度值。
可叠加多个 progress-bar 创建分段进度条。

手风琴折叠 Accordion

Accordion 是可折叠的内容面板，常用于FAQ或分组信息展示，需要 Bootstrap JS 支持。

点击按钮可展开或折叠对应内容，collapsed 类表示初始折叠状态，data-bs-parent 保证手风琴效果，即同时只展开一项。

accordion 是外层容器，accordion-item 是每个折叠项。
accordion-header 包含触发按钮 accordion-button，点击按钮折叠/展开内容。
accordion-collapse collapse 包含折叠内容，可用 show 默认展开。
data-bs-parent="#id" 保证同一时间只有一个折叠项展开。

加载动画 Spinner

Bootstrap 提供内置 Spinner 类，用于显示加载状态或操作等待提示。

提供旋转环和圆点两种类型，内置颜色类直接显示不同状态，可嵌入按钮或表单等界面，提示操作加载中。

spinner-border 创建旋转圆环加载，spinner-grow 创建圆点加载。
text-* 控制颜色。
role="status" 和 span.visually-hidden 用于辅助无障碍访问。

图片辅助类 Image

Bootstrap 提供内置图片辅助类，用于控制图片响应式、圆角或圆形效果。

img-fluid 使图片宽度自适应父容器，高度按比例缩放。
rounded 给图片四个角添加圆角，变体有 rounded-top、rounded-bottom、rounded-start、rounded-end、rounded-circle（圆形）、rounded-pill（椭圆）。
img-thumbnail 给图片添加边框和内边距，显示缩略图效果。

演示不同效果，父容器宽度不同，便于观察响应式

img-fluid（自适应父容器）

rounded（轻微圆角）

rounded-circle（圆形）

img-thumbnail（带边框缩略图）
